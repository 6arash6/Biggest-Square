# ex07: ft_putnbr

**Goal:** Print an integer (handle negatives and INT_MIN).

- **Prototype:** `void ft_putnbr(int nb);`
- **Allowed functions:** Only `write`.
- **How it works:**
  - Uses recursion to print each digit.
  - Handles negatives and INT_MIN edge case.

**Example output:**
```
ft_putnbr(42);   // Output: 42
ft_putnbr(-42);  // Output: -42
```
