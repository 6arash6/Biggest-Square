# ex08: ft_print_combn

**Goal:** Print all unique combinations of n digits in ascending order (n: 1-9).

- **Prototype:** `void ft_print_combn(int n);`
- **Allowed functions:** Only `write`.
- **How it works:**
  - Uses an array to store the current combination.
  - Increments and prints as needed using while loops.

**Example output for n = 2:**
```
01, 02, ..., 89
```
