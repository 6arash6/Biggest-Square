# ex04: ft_is_negative

**Goal:** Print 'N' if the number is negative, 'P' otherwise.

- **Prototype:** `void ft_is_negative(int n);`
- **Allowed functions:** Only `write`.
- **How it works:**
  - Checks if n < 0. If so, prints 'N'. Otherwise, prints 'P'.

**Example output:**
```
ft_is_negative(-5); // Output: N
ft_is_negative(0);  // Output: P
```
