# ex06: ft_print_comb2

**Goal:** Print all combinations of two different two-digit numbers (00 01 ... 98 99).

- **Prototype:** `void ft_print_comb2(void);`
- **Allowed functions:** Only `write`.
- **How it works:**
  - Uses two nested while loops for i and j (i < j).
  - Prints each pair as two two-digit numbers.

**Example output:**
```
00 01, 00 02, ..., 98 99
```
