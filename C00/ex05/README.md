# ex05: ft_print_comb

**Goal:** Print all unique combinations of three different digits in ascending order.

- **Prototype:** `void ft_print_comb(void);`
- **Allowed functions:** Only `write`.
- **How it works:**
  - Uses three nested while loops for a, b, c.
  - Prints only if a < b < c.
  - Adds ", " except after the last combination.

**Example output:**
```
012, 013, ..., 789
```
