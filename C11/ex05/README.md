# ex05: do-op

## Objective
Create a program that performs basic arithmetic operations (+, -, *, /, %) using function pointers.

## Beginner Explanation
- Parse command line arguments.
- Use function pointers for each operator.
- Handle division/modulo by zero.
- Print result using write.

## Example
```
./do-op 42 + 21   # Output: 63
./do-op 25 / 0    # Output: Stop : division by zero
```
