# ex03: ft_div_mod

**Goal:** Divide and modulo two integers, store results in pointers.

- **Prototype:** `void ft_div_mod(int a, int b, int *div, int *mod);`
- **Allowed functions:** None.
- **How it works:**
  - Divides `a` by `b`, stores result in `*div`.
  - Stores remainder in `*mod`.

**Example usage:**
```c
int d, m;
ft_div_mod(10, 3, &d, &m);
// d is 3, m is 1
```
