# ex07: ft_rev_int_tab

**Goal:** Reverse the order of elements in an int array.

- **Prototype:** `void ft_rev_int_tab(int *tab, int size);`
- **Allowed functions:** None.
- **How it works:**
  - Swaps elements from the start and end, moving towards the center.

**Example usage:**
```c
int arr[3] = {1, 2, 3};
ft_rev_int_tab(arr, 3);
// arr is now {3, 2, 1}
```
