# ex08: ft_sort_int_tab

**Goal:** Sort an int array in ascending order.

- **Prototype:** `void ft_sort_int_tab(int *tab, int size);`
- **Allowed functions:** None.
- **How it works:**
  - Uses a simple bubble sort algorithm with while loops.

**Example usage:**
```c
int arr[3] = {3, 1, 2};
ft_sort_int_tab(arr, 3);
// arr is now {1, 2, 3}
```
