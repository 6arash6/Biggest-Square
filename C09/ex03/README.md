# ex03: ft_atoi

## Objective
Write a function that converts a string to an integer.

## Function Prototype
```c
int ft_atoi(const char *str);
```

## Beginner Explanation
- Skip whitespace.
- Handle sign.
- Convert digits.

## Example
```c
ft_atoi("42"); // returns 42
ft_atoi("-42"); // returns -42
```

## Steps
1. Skip whitespace.
2. Handle sign.
3. Convert digits.
