# ex01: ft_putstr

## Objective
Write a function that prints a string to standard output.

## Function Prototype
```c
void ft_putstr(const char *str);
```

## Beginner Explanation
- Use write to print each character.

## Example
```c
ft_putstr("Hello"); // prints Hello
```

## Steps
1. Loop through string.
2. Call write for each character.
