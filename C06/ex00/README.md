# ex00: ft_print_program_name

## Objective
Write a program that prints its own name followed by a newline.

## Beginner Explanation
- Use argv[0] to access the program name.
- Print each character using write.
- End with a newline.

## Example
If compiled as 'a.out':
```
$ ./a.out
./a.out
```

## Steps
1. Loop through argv[0].
2. Print each character.
3. Print newline.

## Manual Testing
Compile and run the program to see its name printed.
