# ex03: ft_params_sum

## Objective
Write a program that prints the number of parameters passed to it.

## Beginner Explanation
- argc contains the number of arguments (including program name).
- Print argc - 1 as a character.

## Example
```
$ ./a.out hello world
2
```

## Steps
1. Calculate argc - 1.
2. Print as a character.
3. Print newline.

## Manual Testing
Compile and run with parameters to see the count printed.
