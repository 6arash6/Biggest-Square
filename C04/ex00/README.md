# C04/ex00: ft_strlen

## Function
`int ft_strlen(char *str);`

## Description
Returns the length of the string (number of characters before the null terminator).

## Example
```
ft_strlen("Piscine42"); // Output: 9
```

## Beginner Tips
- Use a loop to count characters until you reach '\0'.
- Return the count.
