# ex13: ft_list_merge

## Objective
Merge two linked lists.

## Function Prototype
```c
void ft_list_merge(t_list **begin_list1, t_list *begin_list2);
```

## Beginner Explanation
- Find the end of the first list.
- Link it to the second list.

## Example
