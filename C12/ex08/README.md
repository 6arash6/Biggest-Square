# ex08: ft_list_reverse

## Objective
Reverse a linked list.

## Function Prototype
```c
void ft_list_reverse(t_list **begin_list);
```

## Beginner Explanation
- Loop through the list.
- Reverse the next pointers.
- Update the head pointer.

## Example
```c
ft_list_reverse(&head);
```
