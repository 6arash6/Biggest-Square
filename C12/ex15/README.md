# ex15: ft_list_reverse_fun

## Objective
Reverse a linked list in place (fun version).

## Function Prototype
```c
void ft_list_reverse_fun(t_list *begin_list);
```

## Beginner Explanation
- Swap data between elements from start and end.
- Do not change next pointers.

## Example
```c
ft_list_reverse_fun(head);
```
