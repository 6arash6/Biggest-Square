# ex05: ft_list_at

## Objective
Write a function that returns the element at index nbr in a linked list.

## Function Prototype
```c
t_list *ft_list_at(t_list *begin_list, unsigned int nbr);
```

## Beginner Explanation

- Return node at index nbr.

## Example

```c
t_list *node = ft_list_at(head, 1);
```
