# ex06: ft_list_reverse

## Objective
Write a function that reverses a linked list.

## Function Prototype
```c
void ft_list_reverse(t_list **begin_list);
```

## Beginner Explanation
- Loop through the list.
- Reverse next pointers.
- Update begin_list.

## Example
```c
ft_list_reverse(&head);
```

## Steps
1. Loop through list.
2. Reverse pointers.
3. Update begin_list.
